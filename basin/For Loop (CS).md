A loop used to repeat a [[Block (CS)|block]] of code a known number of times.
###### Java

```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```
###### Python

```python
for i in range(5):
    print(i)  # prints 0 to 4
```
###### JavaScript

```javascript
for (let i = 0; i < 5; i++) {
    console.log(i);
}
```
